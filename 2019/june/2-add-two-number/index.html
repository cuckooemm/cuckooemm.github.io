LeetCode - 2 - Add Two Numbers [Medium]

2.Add Two Numbers

地址
难度：Medium

题目描述

You are given two non-empty linked lists representing two non-negative integers. The digits are stored in reverse order and each of their nodes contain a single digit. Add the two numbers and return it as a linked list.

You may assume the two numbers do not contain any leading zero, except the number 0 itself.

给定两个非空链表用来表示两个非负整数，它们各自的位数是按照逆序的方式存储的，并且每个节点只能存储一位数字。用链表返回它们的和。
你可以假设它们都不会以0开头，除了数字0。

示例

Input: (2 -> 4 -> 3) + (5 -> 6 -> 4)
Output: 7 -> 0 -> 8
Explanation: 342 + 465 = 807.

思路

用l1作为结果存放位置，节省空间与内存申请的时间。
之前用第三个链表保存结果，这次写博客又想到了用l1参数自身保存结果，空间复杂度降到了O(1)。
没什么说的，挺简单的中等难度题。
时间复杂度:O(max(l1,l2)), 空间复杂度：O(1)。

代码

/**
 * Definition for singly-linked list.
 * struct ListNode {
 *     int val;
 *     ListNode *next;
 *     ListNode(int x) : val(x), next(NULL) {}
 * };
 */
class Solution {
public:
    ListNode* addTwoNumbers(ListNode* l1, ListNode* l2) {
        int t = 0;                 //保存进位
        ListNode* list = l1;       // 保存头结点
        ListNode* pre;             // 保存l1的上一个节点
        while(l1 != NULL && l2 != NULL){
            t += l1->val + l2->val;
            pre = l1;
            l1->val = t % 10;
            t /= 10;
            l1 = l1->next;
            l2 = l2->next;
        }
        if(l2 != NULL){           // l2 不为NULL 
           pre->next = l2;        // 则把l2接到l1
           l1 = pre->next;        // 之前的l1指针已被更改 需重新赋值
        }
        while(l1 != NULL){
            if(t != 0){           // 循环 存在 1 + 9一直存在进位的情况
                t += l1->val;
                pre = l1;
                l1->val = t % 10;
                t /= 10;
                l1 = l1->next;
            }else{
                return list;     // t=0 直接返回
            }
        }
        if(t != 0){              // 如果l1 l2 同时为NULL 则可能产生进位情况
            pre->next = new ListNode(t);
        }
        return list;
    }
};
