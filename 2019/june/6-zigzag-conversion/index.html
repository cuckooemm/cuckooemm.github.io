6.ZigZag Conversion

地址
难度：Medium

题目描述

The string "PAYPALISHIRING" is written in a zigzag pattern on a given number of rows like this: (you may want to display this pattern in a fixed font for better legibility)
使字符串以给定行数上进行Z字形书写，像下面这样。

P   A   H   N
A P L S I I G
Y   I   R

And then read line by line: "PAHNAPLSIIGYIR"
然后一行一行的读。
Write the code that will take a string and make this conversion given a number of rows:
编写代码。

string convert(string s, int numRows);

示例

Example 1:

Input: s = “PAYPALISHIRING”, numRows = 3
Output: “PAHNAPLSIIGYIR”

Example 2:

Input: s = “PAYPALISHIRING”, numRows = 4
Output: “PINALSIGYAHRPI”
Explanation:

P     I    N
A   L S  I G
Y A   H R
P     I

思路

就是计算字符串下标。

  • 行0中的字符位于索引k(2 * numRows - 2)处。
  • 行numRows-1中的字符位于索引k(2 * numRows − 2) + numRows − 1处;
  • 内部的行i中的字符位于索引k(2 * numRows − 2) + i 以及 (2 * numRows − 2) − i处;
    时间复杂度:O(n), 空间复杂度：O(n)。

AC

  • Runtime: 12 ms, faster than 94.01% of C++ online submissions for ZigZag Conversion.
  • Memory Usage: 10.2 MB, less than 85.33% of C++ online submissions for ZigZag Conversion.

代码

class Solution {
public:
    string convert(string s, int numRows) {
        if(s.size() == 0 | numRows == 1){
            return s;
        }
        string res = "";
        int n = s.size();
        int size = (numRows << 1) - 2;
        int temp = 0;
        for(int i = 0; i < numRows; ++i){
            for(int j = i; j < n; j += size){
                res += s[j];
                temp = j + size - (i << 1);
                if(i != 0 && i != numRows - 1 && temp < n)
                    res += s[temp];
            }
        }
        return res;
    }
};
