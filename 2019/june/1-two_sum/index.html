LeetCode - 1 - Two Sum [Easy]

Written by Cuckooemm with ♥ on June 1, 2019 in algorithm

1.Tow Sum

地址
难度：Easy

题目描述

Given an array of integers, return indices of the two numbers such that they add up to a specific target.

You may assume that each input would have exactly one solution, and you may not use the same element twice.

给定一个整型数组，返回数组中值的和target的数组索引，可以假定每个输入都有一个解，且同一个元素不能使用两次。

示例

Given nums = [2, 7, 11, 15], target = 9,
Because nums[0] + nums[1] = 2 + 7 = 9,
return [0, 1].

思路

  • 空间复杂度优先:
    首先对数组进行排序，左指针指向数组头，右指针指向数组长度-1，左指针+右指针。 如果结果大于target,右指针–，反之左指针++。
    时间复杂度:O(nlogn), 空间复杂度：O(1)。

  • 时间复杂度优先:
    字典法,顺序读取元素，把target-element的值保存字典。
    时间复杂度:O(n), 空间复杂度：O(n)。

AC

  • Runtime: 4 ms, faster than 99.94% of C++ online submissions for Two Sum.

代码

字典法

class Solution {
public:
    vector<int> twoSum(vector<int>& nums, int target) {
	unordered_map<int, int> map(nums.size());
	vector<int> result(2);
	for(int i = 0; i < nums.size(); i++)    // 元素大小为key,value为索引
	{
		map[nums[i]] = i;
	}
	for (int i = 0; i < nums.size(); i++)
	{
		int t = target - nums[i];
		if (map.count(t) && map[t] != i)    // 判断不是同一个元素
		{
			result[0] = i;
			result[1] = map[t];
			break;
		}
	}
	return result;
    }
};
