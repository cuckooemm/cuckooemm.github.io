22.Generate_Parentheses

地址
难度：Medium

题目描述

Given n pairs of parentheses, write a function to generate all combinations of well-formed parentheses.

For example, given n = 3, a solution set is:

给出 n 代表生成括号的对数，请你写出一个函数，使其能够生成所有可能的并且有效的括号组合。

示例

例如，给出 n = 3，生成结果为：

[
  "((()))",
  "(()())",
  "(())()",
  "()(())",
  "()()()"
]

思路

使用递归。

AC

  • Runtime: 4 ms, faster than 96.72% of C++ online submissions for Generate Parentheses.
  • Memory Usage: 17.5 MB, less than 22.52% of C++ online submissions for Generate Parentheses.

代码

class Solution {
private:
    void generate(string s,int l,int r,int n){
        if(s.size() == n << 1){
            res.push_back(s);
            return;
        }
        if(l < n)
            generate(s + '(',l + 1,r,n);
        if(r < l)
            generate(s + ')',l,r + 1,n);
    }
public:
    vector<string> res;
    vector<string> generateParenthesis(int n) {
        generate("",0,0,n);
        return res;
    }
};
