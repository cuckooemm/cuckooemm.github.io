50.Pow(x,n)

地址
难度：Medium

题目描述

Implement pow(x, n), which calculates x raised to the power n (xn).

示例

Example 1:

Input: 2.00000, 10
Output: 1024.00000

Example 2:

Input: 2.10000, 3
Output: 9.26100

Example 3:

Input: 2.00000, -2
Output: 0.25000
Explanation: 2-2 = 1⁄22 = 1⁄4 = 0.25

思路

快速幂。

AC

  • Runtime: 4 ms, faster than 72.85% of C++ online submissions for Pow(x, n).
  • Memory Usage: 8.5 MB, less than 21.03% of C++ online submissions for Pow(x, n).

代码

class Solution {
public:
    double myPow(double x, int n) {
        double res = 1.0;
        for (int i = n; i != 0; i /= 2) {
            if (i & 1) res *= x;
            x *= x;
        }
        return n < 0 ? 1 / res : res;
    }
};
