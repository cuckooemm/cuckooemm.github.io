42.Trapping Rain Water

地址
难度：Hard

题目描述

Given n non-negative integers representing an elevation map where the width of each bar is 1, compute how much water it is able to trap after raining.

给定 n 个非负整数表示每个宽度为 1 的柱子的高度图，计算按此排列的柱子，下雨之后能接多少雨水。

示例

Example:

Input: [0,1,0,2,1,0,1,3,2,1,2,1]
Output: 6

思路

使用双指针指向两端，积水高度由两边指针决定。
遍历时维护lmax和rmax，如果当前高度低于*max，则计算积水量。
如果当前高度高于或等于*max，则更改lmax(rmax)。

AC

  • Runtime: 8 ms, faster than 72.98% of C++ online submissions for Trapping Rain Water.
  • Memory Usage: 9 MB, less than 86.30% of C++ online submissions for Trapping Rain Water.

代码

class Solution {
public:
    int trap(vector<int>& height) {
        int l = 0, r = height.size() - 1;
        int res  = 0;
        int lmax = 0, rmax = 0;
        while(l < r){
            if(height[l] < height[r]){
                if(height[l] >= lmax)
                    lmax = height[l];
                else
                    res += lmax - height[l];
                ++l;
            }else{
                if(height[r] >= rmax)
                    rmax = height[r];
                else
                    res += rmax - height[r];
                --r;
            }
        }
        return res;
    }
};
